# **Hisobotlar — Umumiy ko'rinish**

Biznex POS hisobotlar tizimi biznesingizning moliyaviy va operatsion holatini **real vaqtda** kuzatish imkonini beradi. Barcha savdo, xarid, xodim va ombor ma'lumotlari bir joyda jamlangan.

---

## **Hisobotlar nima uchun kerak?**

1. **Foyda va zararni bilish** — kunlik, haftalik, oylik daromad ko'rinadi.
2. **Xarajatlarni nazorat qilish** — qayerga qancha pul ketayotganini kuzatish.
3. **Xodimlar samaradorligi** — qaysi xodim qancha savdo qilganini bilish.
4. **Mahsulot tahlili** — qaysi mahsulot ko'p sotilayotganini aniqlash.
5. **Qaror qabul qilish** — ma'lumotlarga asoslangan biznes qarorlari.

---

## **Hisobot turlari**

### Asosiy hisobotlar (Monitoring)

| Hisobot | Tavsif |
|---------|--------|
| **Umumiy monitoring** | Kunlik savdo, tushum, to'lov turlari |
| **Buyurtmalar** | Buyurtmalar soni, summasi, holatlari |
| **To'lovlar** | To'lov turlari bo'yicha tahlil (naqd, karta, onlayn) |
| **Mahsulotlar** | Eng ko'p va kam sotiladigan mahsulotlar |
| **Masalliqlar** | Masalliq sarfi va qoldiqlari |
| **Xodimlar** | Har bir xodimning savdo ko'rsatkichlari |
| **Kategoriyalar** | Kategoriya bo'yicha savdo tahlili |
| **Bekor qilishlar** | Bekor qilingan buyurtmalar tahlili |
| **Tranzaksiyalar** | Moliyaviy operatsiyalar tarixi |
| **Xaridlar** | Xarid operatsiyalari |
| **Fiskal** | Fiskal cheklar va Z-hisobotlar holati, OFD sinxronizatsiya |

### Tahlil vositalari

| Hisobot | Tavsif |
|---------|--------|
| **ABC tahlil** | Pareto asosida masalliqlar muhimlik tahlili |
| **Food Cost** | Retsept rentabelligi va tannarx hisoblash |
| **Avtomatik xarid** | Minimal zaxira asosida xarid ro'yxati generatsiyasi |
| **Iste'mol hisoboti** | Masalliq sarflanish tahlili |
| **Narx tarixi** | Masalliq narxlari o'zgarishi |
| **Zaxira hisoboti** | Omborxona qoldiqlari holati |
| **Isrof hisoboti** | Yo'qotilgan va muddati o'tgan mahsulotlar |

### Oshxona ishlab chiqarish

| Hisobot | Tavsif |
|---------|--------|
| **Mahsulot → Retsept** | Kerakli masalliq miqdorini hisoblash |
| **Retsept → Mahsulot** | Mavjud masalliqdan maksimal ishlab chiqarish |

---

## **Hisobotlarga kirish**

1. Biznex POS **bosh menyusida** "Hisobotlar" bo'limini tanlang
2. Kerakli hisobot turini tanlang
3. **Sana oralig'ini** belgilang (bugun, hafta, oy, maxsus)
4. Hisobot ekranda ko'rsatiladi

:::tip
Hisobotlarni **Biznex Cloud** orqali masofadan ham ko'rish mumkin — [client.biznex.uz](https://client.biznex.uz)
:::

---

## **Maslahatlar**

- **Har kuni tekshiring:** Kunlik hisobotni ko'rib chiqish — muammolarni erta aniqlashga yordam beradi
- **Haftalik tahlil:** Har hafta oxirida umumiy ko'rsatkichlarni taqqoslang
- **ABC tahlil:** Oyiga kamida 1 marta — A kategoriya masalliqlariga alohida e'tibor bering
- **Food Cost:** 40% dan oshsa — retseptni yoki narxni qayta ko'rib chiqing
- **Xodim motivatsiyasi:** Eng yaxshi ko'rsatkichga ega xodimlarni rag'batlantiring
